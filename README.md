# 📊 Customer Prediction SaaS Dashboard

Welcome to **AlphaDay15_SaaS_Dashboard**, an interactive **Streamlit-based dashboard** powered by AI to predict customer buying behavior.  

This project combines **Python AI (Logistic Regression)** and a **web-based UI** into a full SaaS dashboard, where users can adjust sliders to simulate customer data and get live predictions.

---

## 🔹 Features

- **Interactive Sliders:**  
  Adjust customer age and minutes on site to generate predictions.

- **Live AI Prediction:**  
  Logistic Regression model predicts whether the customer will buy a product.

- **Celebration Animation:**  
  Balloons animation appears for customers predicted to buy.

- **User-Friendly UI:**  
  Simple, responsive dashboard built with Streamlit.

---

## 🛠️ Technologies Used

- **Python**  
- **Streamlit** (Web UI)  
- **Pandas** (Data Handling)  
- **Scikit-Learn** (Logistic Regression)
